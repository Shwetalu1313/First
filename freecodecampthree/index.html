HTML

HTML stands for HyperText Markup Language. It is the standard markup language for creating websites and web applications.

Sample HTML code:

<html> <head> <title>Page Title</title> </head> <body> <h1>Hello, World!</h1> </body> </html>
  • Use <h1> to define headings
  • Use <p> to define paragraphs
  • Use <a> to create links
CSS

CSS stands for Cascading Style Sheets. It is used for styling the layout and appearance of web pages.

Sample CSS code:

body {  background-color: #f2f2f2;  font-family: Arial, sans-serif;  color: #333;  margin: 0;  padding: 0; }
  • Use CSS selectors to target HTML elements
  • Use properties like background-color and color to style elements
  • Use @media queries for responsive design
